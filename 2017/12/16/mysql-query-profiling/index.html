　　如何解读这份报告呢？

  • Response：查询的响应时间。
  • time：查询占总时间的百分比。
  • Calls：查询的执行次数。
  • R/Call：单次查询的平均响应时间。
  • V/M：方差与平均值的比值。如果这个值较大，表示查询对应的执行时间变化较大。
